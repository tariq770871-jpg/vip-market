# VIP MARKET GLOBAL

**VIP MARKET GLOBAL** هي منصة تداول متقدمة وحصرية مصممة للمتداولين المحترفين والنخبة. توفر المنصة أدوات تحليل متطورة، أكاديمية تدريبية شاملة، وخزنة آمنة لحماية الأصول الرقمية.

## المميزات الرئيسية

- **لوحة تحكم متقدمة**: واجهة مستخدم حديثة وسهلة الاستخدام مع تصميم داكن احترافي
- **نظام المصادقة الآمن**: تسجيل دخول وإنشاء حسابات محمية بتشفير قوي
- **أكاديمية تدريبية**: دروس احترافية حصرية للمتداولين
- **أدوات الذكاء الاصطناعي**: تحليل متقدم وتنبؤات السوق
- **الخزنة المؤمنة**: حماية عالية للأصول الرقمية بمعايير أمان عالمية
- **معرض المنتجات**: عرض شامل للمنتجات والخدمات المتاحة
- **نموذج الاتصال**: تواصل مباشر مع فريق الدعم

## المتطلبات

- Node.js >= 18.0.0
- npm >= 9.0.0
- متصفح حديث (Chrome, Firefox, Safari, Edge)

## التثبيت والتشغيل

### 1. استنساخ المستودع

```bash
git clone https://github.com/tariq770871-jpg/vip-market.git
cd vip-market
```

### 2. تثبيت الاعتماديات

```bash
npm install
```

### 3. إعداد متغيرات البيئة

انسخ ملف `.env.example` إلى `.env` وأضف بيانات Supabase الخاصة بك:

```bash
cp .env.example .env
```

ثم عدّل الملف بإضافة:

```env
VITE_SUPABASE_URL=https://vclsqisvkyalshvpsqkd.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key_here
```

### 4. تشغيل خادم التطوير

```bash
npm run dev
```

سيتم فتح المشروع على `http://localhost:5173`

### 5. البناء للإنتاج

```bash
npm run build
```

## الملفات الرئيسية

### main.js
يحتوي على:
- إدارة القائمة الجانبية (Sidebar)
- معالجة النماذج والتحقق من الصحة
- تحسين الصور
- إدارة إمكانية الوصول

### auth.js
يحتوي على:
- إدارة المصادقة (تسجيل الدخول والتسجيل)
- إدارة الجلسات
- التحقق من بيانات المستخدم

### supabase.js
يحتوي على:
- اتصال قاعدة البيانات Supabase
- عمليات CRUD
- التحقق من البيانات

### security.js
يحتوي على:
- تنظيف المدخلات (Input Sanitization)
- منع هجمات XSS
- مراقبة الأداء
- تحسين الصور والموارد

## الأمان

- **تشفير البيانات**: جميع البيانات الحساسة مشفرة باستخدام Supabase
- **منع XSS**: تنظيف جميع المدخلات من المستخدمين
- **CSRF Protection**: حماية من هجمات CSRF
- **Rate Limiting**: تحديد معدل الطلبات لمنع الإساءة
- **متغيرات البيئة**: عدم تخزين المفاتيح السرية في الكود

## الأداء

- **Lazy Loading**: تحميل الصور بطريقة كسولة
- **Minification**: تصغير ملفات CSS و JavaScript
- **Caching**: تخزين مؤقت للموارد
- **CDN**: استخدام CDN للخطوط والمكتبات الخارجية
- **Performance Monitoring**: مراقبة أداء الصفحة
